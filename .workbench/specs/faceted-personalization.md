# Claude Notes (2026-07-19)

Спека изучена досконально. Всё ясно. Готов к реализации когда дойдём до соответствующих фаз.

## Ключевые решения зафиксированы:

1. **Профиль для всех** — принято. Для обучения и корректировки алгоритмов. consent-gated остаётся для аналитики/рекламы.

2. **impression_agg** (не log) — одна строка per (user, venue) с UPDATE. Принято из ревью.

3. **Калибровка с гейтом** — F2.5 (streak+epsilon сразу) → F2.6 (Steck только на зрелых профилях ≥10 сигналов). Принято из ревью.

4. **Уточняющая карточка на 5-6** с композитным триггером (позиция + признак трения). Принято из ревью.

5. **"Спланируй день" поля** — заполняются в Gemini-проходе (A8), но логика отложена на Future. Просто иметь в виду.

6. **JSONB профиля** — ок для MVP. Профиль парсится 1 раз за запрос, facet_* на venues уже нативные text[].
