# LaziGo Product Brief

**Contextual leisure discovery for Tbilisi. Not a map — a decision engine.**

## Core Value

Google Maps tells you WHAT exists. LaziGo tells you WHERE TO GO.

1. **Compound context** — 5+ dimensions simultaneously (interest + company + pet + time + distance)
2. **Explainability** — "Тебе нравится: природа. Открыто. Для пары. 11 мин"
3. **Decision reduction** — 8 scored results, not 200
4. **Events + places unified** — Google Events and Maps are separate, we merge them
5. **Social context** — "I'm with family + dog" changes everything

## Coverage (Tbilisi)

- 3,168 venues (OSM)
- ~68 events (5 sources, 3 active)
- 1,755 Google-matched with ratings
- 3 languages: ru, en, ka

## User Journey

1. Landing → interest chips + company
2. Discover → 8 scored cards with explanations
3. Interact → save/hide teaches taste profile
4. Return → personalized results from 2nd visit

## Moat Timeline

```
Month 1-3:  Intelligence (scoring, explanations, context) — DONE
Month 3-6:  Behavioral data (saves/hides from real users)
Month 6-12: Community (tips, collections, badges)
Month 12+:  Curator network + multi-city
```
